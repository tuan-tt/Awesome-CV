## How to Use

#### Requirements

```bash
sudo apt-get install texlive-full
sudo apt-get install texlive-fonts-recommended texlive-fonts-extra
sudo apt-get install fonts-font-awesome
```

#### Usage

```bash
xelatex resume.tex
```

This should result in the creation of ``resume.pdf``
